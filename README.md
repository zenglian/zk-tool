# zk-tool
tool for managing helm chart kafka+zookeeper

## Folders
kafka/       -- helm chart of kafka
zookeeper/   -- zookeeper chart of kafka
zktool/      -- admin and test tool 

## Usage

`zktool/admin-zk.sh`  

`zktool/test-kafka.sh`
